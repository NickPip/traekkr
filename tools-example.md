# Example tool – copy into Admin → Tools

Use these values to create your first tool in **Admin → Collections → Tools → Create New**.

---

## Example: Wireshark

| Field | Value |
|-------|--------|
| **Heading** | Wireshark |
| **Slug** | wireshark |
| **Date** | 2025-02-23 (or today) |
| **Link** | https://github.com/wireshark/wireshark |
| **Description** | Wireshark is the world’s most widely used network protocol analyzer. It lets you capture and interactively browse traffic on a live network or from a capture file. Essential for network troubleshooting, analysis, and security research. |
